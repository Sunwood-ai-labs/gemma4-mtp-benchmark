Gemma 4 MTP Benchmark

A small reproducible benchmark for comparing Gemma 4 LiteRT-LM generation with Multi-Token Prediction enabled and disabled.

GitHub repository · 日本語 README

Quick Start

git clone https://github.com/Sunwood-ai-labs/gemma4-mtp-benchmark.git
cd gemma4-mtp-benchmark
uv venv
uv pip install -e '.[bench]'
uv run gemma4-mtp-bench run --model e2b --backend gpu --prompt-set quick
E2B

Best first run for 8 GB or more Apple Silicon Macs.

E4B

More interesting on 16 GB or larger Macs, especially for coding prompts.

MTP Compare

Runs the same prompt set with speculative decoding off and on.

Task Matrix

Compare coding, JSON formatting, extraction, rewriting, translation, and creative prompts.

Local M1 Max Snapshot

In a one-round local run, predictable tasks showed the clearest gains: E4B GPU coding reached about 1.87x, JSON formatting about 1.65x, and log extraction about 1.51x. Creative and very short prompts were flat or slower.
